#  App RachaAi - Dashboard Financeiro

Projeto desenvolvido para a disciplina de Interação Humano Computador e UX utilizando Blazor Web App com C# e .NET 8.

---

#  Objetivo da Atividade

Desenvolver uma página Dashboard para o aplicativo fictício “RachaAi”, aplicando conceitos de:

- Hierarquia Visual
- Componentização
- Responsividade
- Heurísticas de UX
- Feedback visual
- Organização de interface

---

#  Funcionalidades Implementadas

##  Dashboard Financeiro

A aplicação possui uma tela principal contendo:

- Total a Receber
- Total a Pagar
- Saldo Geral

Os cards utilizam cores para melhorar a visualização financeira do usuário:

- Verde → crédito
- Vermelho → débito
- Preto → saldo geral
